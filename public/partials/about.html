About

Welcome to our first group app! It was created by James Bennett, Ashley Brickhouse, and Justin Warren. You can learn more about us below.


As our Unit 3 group project, we decided to create an app that we thought would be useful. Fit.js is a fitness tracker app that provides a place for you to create and store workouts. Our goal was to build an intuitive app that is simple to use and functional.

Some additions we would like to make include:

-Relational Databases: This improvement would allow the user to save their workouts to their individual username by incorporating a Users model that would store information based on the workouts created by that user. We would also like to create the ability for a user to create more detailed workouts by adding an Exercises model.

-Carousel: We thought that a carousel would also be a great feature in order to scroll through the workout videos on the Videos page or on the welcome page when the user logs in (for inspiration).

-Modal: A modal would make creating more detailed workouts a lot cleaner. The difference would be that when a user clicks on the New Workout tab and creates a workout with a title and duration, they would then see a modal pop up that would give them the ability to enter sets, reps and rest time for each individual exercise.

Ultimately, our priority for this project was to gain experience working with Git and navigating our way through working in a group to conceptualize, organize, and deliver the results of the collaboration.
